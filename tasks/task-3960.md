# Task 3960: AI FIRE (Financial Independence) Calculator

**App Name:** ai-fire-calculator
**Priority:** HIGH
**Theme:** AI × Personal Finance, Budgeting & Money Management

## App Description
A Next.js 16 app that calculates FIRE (Financial Independence, Retire Early) numbers and generates a roadmap. Users input current savings, spending, income; the AI calculates FIRE number (25x annual expenses), time to FIRE, and monthly savings needed.

## Implementation Details
- Form inputs: annual income, annual spending, current savings, investment return expectation, withdrawal rate preference
- Output: FIRE number, years to FIRE, monthly savings required, coast FIRE age, barista FIRE option
- UI: Dark gradient with pink (HSL 300,70%,60%) accent
- API: DeepSeek via /api/generate route with FIRE planning prompt
- Build: `npm run build` must pass
