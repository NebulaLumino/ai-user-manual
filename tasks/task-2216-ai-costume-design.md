# Task 2216: AI Costume Design & Period Wardrobe Generator (Film/TV/Theatre)
- **Status**: [✅] Not Started
- **Priority**: MEDIUM
- **Theme**: AI x Fashion, Apparel Design & Sustainable Style
- **App Dir**: ai-costume-design
- **GitHub**: https://github.com/NebulaLumino/ai-costume-design
- **Description**: Generate costume design concepts for film, TV, or theatre productions based on character profiles, period, and budget
- **Form Fields**: production type (film/TV/theatre), time period/era, genre/tone, character name and brief description, number of costume pieces needed, budget per costume, production scale (indie/Studio), actor measurements if known
- **Output**: Costume design concept description per character, silhouette and key silhouette notes, fabric/textile suggestions with period accuracy, color palette per character and how it communicates arc, breakdown of costume pieces (principal/background/extras), rental vs. build vs. pull recommendations, costume plot (scene-by-scene changes)
