- **App Dir**: ai-flashcard-deck
- **GitHub**: https://github.com/NebulaLumino/ai-flashcard-deck
- **Description**: Generate flashcard decks from any topic — automatically optimized for spaced repetition learning
- **Form Fields**: topic or source material, number of cards desired, card complexity (simple/advanced), subject area, target retention timeline (exam in X weeks), format preference (text-only/image-included/both), learner's prior knowledge
